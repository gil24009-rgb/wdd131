NPS Park Explorer – Site Plan

Purpose

This site provides a simplified, responsive interface for exploring Yellowstone National Park using live data from the National Park Service API. The goal is to present critical information such as alerts, visitor services, and activities in a clean and structured layout.

Target Audience

  • Visitors planning a trip to Yellowstone
  • Families checking current conditions
  • Mobile users needing quick park updates
  • Students reviewing API driven web development

Dynamic Elements Description

The site uses JavaScript modules and the NPS API to dynamically render content.

  • Fetch park data using async and await
  • Fetch alerts using park code parameter
  • Render visitor centers dynamically
  • Render activities list from API data
  • Mobile navigation toggle with aria-expanded updates
  • Accordion behavior using details element

Logo

The logo uses a geometric triangular shape inspired by the existing site styling. It reflects the National Park theme through minimal structure and strong contrast.
